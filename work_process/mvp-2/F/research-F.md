# Research F: localLLM 연결 확인

## 작업 목표
- 설정된 baseURL을 통해 로컬 LLM 서버와의 통신 가능 여부를 확인하는 로직 설계.

## 주요 연구 내용
1. **baseURL 입력 및 처리**:
   - 사용자로부터 입력받은 URL의 유효성 검사 및 `/v1` 접미사 처리.
2. **연결 확인 요청**:
   - `/models` 또는 `/v1/models` 엔드포인트에 GET 요청을 보내 서버 상태 확인.
3. **실패 및 Timeout 처리**:
   - 서버가 꺼져 있거나 응답이 늦을 경우(Timeout)에 대한 예외 처리 로직.
4. **사용자 안내**:
   - 연결 실패 시 서버 구동 여부 확인을 요청하는 명확한 에러 메시지 구성.
