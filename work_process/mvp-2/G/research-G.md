# Research G: localLLM 모델 목록 조회

## 작업 목표
- 로컬 LLM 서버에서 사용 가능한 모델 리스트를 가져오는 기능 설계.

## 주요 연구 내용
1. **API 호출**:
   - `/v1/models` 엔드포인트를 호출하여 JSON 응답 수신.
2. **JSON 파싱 및 추출**:
   - 응답 데이터에서 `data` 배열 내부의 각 객체의 `id` 필드를 추출하여 모델명 리스트 생성.
3. **오류 응답 처리**:
   - 응답 형식이 표준 OpenAI API와 다르거나 빈 목록이 올 경우의 예외 처리.
4. **목록 반환 구조**:
   - UI 단계에서 사용할 수 있도록 문자열 배열 형태로 반환하는 `listModels(config)` 함수 설계.
