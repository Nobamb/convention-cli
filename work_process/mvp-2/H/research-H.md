# Research H: 로컬 모델 선택 UI

## 작업 목표
- 조회된 모델 목록을 사용자에게 보여주고 하나를 선택하게 하는 인터랙티브 UI 설계.

## 주요 연구 내용
1. **prompts select UI 구현**:
   - `prompts` 라이브러리를 사용하여 키보드 방향키로 선택 가능한 리스트 UI 구성.
2. **choices 변환**:
   - `listModels`에서 받은 문자열 배열을 `prompts`에서 요구하는 `{ title, value }` 형식의 객체 배열로 변환.
3. **설정 저장**:
   - 사용자가 선택한 모델명을 `config.json`의 `modelVersion` 필드에 저장하는 로직.
4. **사용자 경험(UX)**:
   - 모델이 하나뿐일 때의 처리(자동 선택 등) 및 선택 완료 후 안내 메시지.
